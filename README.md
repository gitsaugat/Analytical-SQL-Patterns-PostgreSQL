# Pagila Advanced SQL Analytics

This repository demonstrates advanced SQL analytics using the Pagila PostgreSQL dataset.

## Skills Demonstrated

- Multi-table joins
- Window functions (ROW_NUMBER, RANK, LAG)
- Sequential pattern detection
- Revenue share and Pareto analysis
- Partitioned cumulative calculations
- Grain control and aggregation discipline

## Dataset
PostgreSQL Pagila sample database.

## Highlights
- Customer lifetime value calculation
- Month-over-month growth
- Top N per partition
- Consecutive pattern detection
- Revenue ranking within category
